# Anna Nam Lab

This repository contains the Anna Nam Lab website for GitHub Pages.

The Anna Nam Lab studies blood stem cells, clonal hematopoiesis, and the molecular origins of blood cancers using single-cell and multi-omics approaches.

## Website Structure

- `index.html` - homepage
- `research/index.html` - research overview
- `team/index.html` - current members and alumni
- `publications/index.html` - selected publications
- `funding/index.html` - funding agencies
- `labnews/index.html` - lab news and awards
- `contact-us/index.html` - joining and contact information
- `assets/css/site.css` - shared styling and spacing controls
- `assets/images/` - site images with readable filenames

Open `index.html` directly or serve this directory locally with:

```bash
python3 -m http.server
```
